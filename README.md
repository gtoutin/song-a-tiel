# song-a-tiel
Lookup songs and bands and discover new ones


## Endpoints
- [ ] /artist/{name}/similar
- [ ] /artist/{name}/info
- [ ] /song/{name}/similar
- [ ] /song/{name}/info
- [ ] /song/{name}/lyrics

## API powered by Flask
- Compatible with a frontend
- Dockerized
